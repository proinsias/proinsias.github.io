Find a list of all python modules installed on a machine by running the following command
in a terminal:

```python
pydoc modules
```

or:

```python
help('modules')
```

inside of a python interactive shell.

Via [enki.com](https://app.enkipro.com/#/insight/55ed77e3ed2a613a005bddc9).
