Reverse a string with the `rev` command.

```shell
$ echo 'test' | rev
tset
```

It also works with files.

```shell
$ rev Procfile
br.amup/gifnoc C- amup cexe eldnub :bew
```

Via [jwworth/til](https://github.com/jwworth/til).
